# 🛒 API RESTful de E-commerce

Este projeto é uma API backend desenvolvida com Node.js e Express para gerenciamento de usuários, produtos e categorias.

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- Sequelize
- PostgreSQL
- JWT (Autenticação)
- Swagger (Documentação)
- Jest + Supertest (Testes)

## 📦 Funcionalidades

- 👤 Cadastro de usuários
- 🛒 CRUD de produtos
- 🗂️ CRUD de categorias
- 🔐 Rotas protegidas com JWT
- 📄 Documentação com Swagger
- 🧪 Testes automatizados

## ⚙️ Como rodar o projeto

```bash
npm install
npm run dev
